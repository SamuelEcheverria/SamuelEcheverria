## Proyectos Destacados

### 1. Jatif - Sistema de Gestión de Inventario Inteligente
* **Descripción:** Aplicación diseñada para un emprendimiento local enfocada en el registro de precios y el reabastecimiento automatizado de ingredientes. El sistema genera gráficos de analítica comercial interactivos con las tendencias de los productos más vendidos para optimizar estrategias de marketing. Adicionalmente, cuenta con un módulo de proyección de stock que calcula de forma predictiva la fecha estimada de desabastecimiento de cada insumo.
* **Tecnologías utilizadas:** Python (Análisis de datos, lógica del backend y generación de gráficos).
* **Repositorio de código:** [Ver Proyecto Jatif](https://github.com/SamuelEcheverria/jatif)

### 2. Portafolio Web & Currículum Interactivo
* **Descripción:** Hoja de vida digital estructurada de manera semántica. El sitio implementa componentes multimedia avanzados, incluyendo la incrustación de un mapa interactivo para indicar disponibilidad geográfica Y un formulario de contacto directo.
* **Tecnologías utilizadas:** HTML Semántico, CSS.
* **Repositorio de código:** [Ver Proyecto Currículum](https://github.com/SamuelEcheverria/curriculum)

### 3. Sistema de Control de Informes Médicos
* **Descripción:** Solución orientada a la gestión, organización y digitalización de reportes y expedientes del sector salud. El proyecto busca estructurar de manera óptima el flujo de datos médicos para agilizar las consultas informativas y el procesamiento seguro de registros clínicos.
* **Tecnologías utilizadas:** R
* **Repositorio de código:** [Ver Proyecto Informe Médico](https://github.com/SamuelEcheverria/informe_medico/)
